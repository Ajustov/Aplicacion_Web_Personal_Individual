# Mi Sitio Web Personal - Alessandro Justo

Proyecto “Aplicación Web Personal”

## Requisitos
- HTML
- CSS
- JavaScript
- Python
- MySQL Server (Wampserver recomendado)
- Librería: `pip install mysql-connector-python`

## Configuración
1. Importar la base de datos `contactos_db` y crear la tabla `mensajes`.
2. Configurar las credenciales en `server.py` (línea 8).
3. Ejecutar el servidor: `python server.py`.
4. Acceder a: `http://localhost:8000`.

## Funcionalidades Clave
- Formulario con guardado en DB.
- Minijuegos interactivos.
- Panel de administración protegido.
